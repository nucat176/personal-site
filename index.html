Rezonance

Rezonance is a web application for uploading and streaming electronic music, designed using Ruby on Rails and React.js/Redux. Upon entering the site, the user is immersed by the look and feel of a live concert, created using hand-rolled CSS. The navigation is seamless, with continuous playback of songs enabled through the application's single-page architecture. The added functionality of creating, updating and deleting tracks - as well as commenting on tracks - was inspired by SoundCloud.

Marvel Encyclopedia

Marvel Encyclopedia is a front-end web application that interacts with the official Marvel Comics API to create the ultimate resource for finding and learning about the Marvel Comics Universe. Marvel Encyclopedia interacts with the Marvel API for character pictures, names and descriptions, while powers and abilities are obtained by scraping the web.

Flexbox Fox

Flexbox Fox is a web tutorial application inspired by Flexbox Froggy for learning CSS Flexbox. This is a purely front-end application, created using Javascript and React, and a clever implementation of DOM manipulation allows the user to input CSS Flexbox commands in order to guide a group of foxes to their dens. The utilization of React Router allows for the user to easily navigate through 13 levels using a wide variety of Flexbox commands.
